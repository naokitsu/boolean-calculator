# Boolean Calculator
Operations:
- Binary
  - And          - `&`
  - Or           - `V`
  - Implication  - `>`
  - Reversed implication - `<`
  - Xor          - `+`
  - Eq           - `=`
  - Nand         - `|`
  - Nor          - `^`
- Unary
  - Neg          - `~`
- Miscellaneous
    - LeftBracket  - `(`
    - RightBracket - `)`
    - Variables - `a`-`z`
    - Constants - `0`, `1`
